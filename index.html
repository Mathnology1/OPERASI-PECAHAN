üìö Materi: Operasi Pecahan

1. Pecahan Campuran

Pecahan campuran adalah gabungan bilangan bulat dan pecahan biasa.

Contoh: $6\frac{2}{5}$ dibaca "enam dua per lima"

Untuk mengubah ke pecahan biasa: $6\frac{2}{5} = \frac{(6 \times 5) + 2}{5} = \frac{32}{5}$

2. Perkalian Pecahan

Untuk mengalikan pecahan, kalikan pembilang dengan pembilang dan penyebut dengan penyebut.

Rumus: $\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}$

Contoh: $\frac{2}{3} \times \frac{4}{5} = \frac{2 \times 4}{3 \times 5} = \frac{8}{15}$

3. Pembagian Pecahan

Untuk membagi pecahan, kalikan dengan kebalikan (reciprocal) dari pecahan pembagi.

Rumus: $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$

Contoh: $\frac{3}{4} \div \frac{2}{5} = \frac{3}{4} \times \frac{5}{2} = \frac{15}{8}$

4. Urutan Operasi

Dalam operasi campuran, ikuti urutan: Kurung ‚Üí Perkalian/Pembagian ‚Üí Penjumlahan/Pengurangan

Ingat: KALI BAGI dulu, baru TAMBAH KURANG

üßÆ Soal dan Pembahasan

Soal:

Nilai dari $\frac{5}{8} \times 6\frac{2}{5} + 3\frac{4}{5} \div \frac{7}{10}$ adalah ...

A. $8\frac{5}{7}$
B. $8\frac{6}{7}$
C. $9\frac{1}{2}$
D. $9\frac{3}{7}$
E. $10\frac{1}{5}$

üìù Pembahasan Langkah demi Langkah

Langkah 1: Ubah pecahan campuran ke pecahan biasa

$6\frac{2}{5} = \frac{(6 \times 5) + 2}{5} = \frac{30 + 2}{5} = \frac{32}{5}$

$3\frac{4}{5} = \frac{(3 \times 5) + 4}{5} = \frac{15 + 4}{5} = \frac{19}{5}$

Langkah 2: Tulis ulang soal

$\frac{5}{8} \times \frac{32}{5} + \frac{19}{5} \div \frac{7}{10}$

Langkah 3: Hitung perkalian pertama

$\frac{5}{8} \times \frac{32}{5} = \frac{5 \times 32}{8 \times 5} = \frac{160}{40} = 4$

Langkah 4: Hitung pembagian

$\frac{19}{5} \div \frac{7}{10} = \frac{19}{5} \times \frac{10}{7} = \frac{19 \times 10}{5 \times 7} = \frac{190}{35} = \frac{38}{7}$

Langkah 5: Jumlahkan hasil

$4 + \frac{38}{7} = \frac{28}{7} + \frac{38}{7} = \frac{66}{7}$

Langkah 6: Ubah ke pecahan campuran

$\frac{66}{7} = 9\frac{3}{7}$ (karena $66 \div 7 = 9$ sisa $3$)

‚úÖ Jawaban: D. $9\frac{3}{7}$
